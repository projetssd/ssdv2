Simple Seedbox docker

Français Hosted on GitHub Pages — Theme by mattgraham



Le script SSD permet l’installation d’une seedbox complète avec Docker

Pour se lancer dans l’installation, il faut au préalable suivre les étapes suivantes:

  • Un serveur distant
  • Un nom de domaine
  • Un abonnement à cloud workspace
  • DNS Cloudflare
  • Création API Google Drive
  • [Création remote Rclone] Pour MacOS Pour Windows Pour Linux

###

Une fois les pré requis validé pous pouvez suivre le guide :

  • Pas à pas d’installation
